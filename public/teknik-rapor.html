Kullanılan Teknolojiler

Bu site herhangi bir React, Vue veya Angular gibi modern framework kullanmadan, tamamen saf (Vanilla) JavaScript ES Modülleri ile yazılmıştır. Aşağıda her katmanın rolü açıklanmaktadır.

Build Tool
⚡ Vite
Geliştirme sunucusu ve modül bundler. Dosya değişikliklerini anında tarayıcıya yansıtır (HMR). ?raw import desteği ile CSS dosyalarını JavaScript string olarak içe aktarır.
CSS Framework
🎨 Tailwind CSS (CDN)
Utility-first CSS kütüphanesi. HTML içinde doğrudan sınıf isimleriyle stil verilir. PostCSS yerine tarayıcı içi runtime modu kullanılmıştır.
Language
📦 ES Modules (ESM)
Her bileşen ayrı bir .js dosyasıdır. import/export ile birbirine bağlanır. Tarayıcı native module desteği kullanılır.
Typography
🔤 Google Fonts — Outfit
"Outfit" font ailesi, 300–800 ağırlık aralığıyla tüm tipografi ihtiyaçlarını karşılar. Fiyat göstergelerinde "Courier New" monospace kullanılır.
Performance
🎞️ requestAnimationFrame
Bloomberg ticker animasyonu CSS yerine RAF (requestAnimationFrame) ile yürütülür. Bu sayede 60fps akıcı kaydırma sağlanır ve tarayıcı önbelleğinden etkilenmez.
Offline
🔧 Service Worker
PWA (Progressive Web App) desteği için sw.js kullanılır. Cache versiyonlama ile eski dosyaların önbellekte kalması önlenir.
Storage
💾 localStorage API
Favoriler ve wizard cevapları tarayıcının yerel depolama alanına kaydedilir. Sayfa yenilendiğinde veriler korunur.
Graphics
📐 SVG (inline)
Radar Chart (6 eksenli örümcek grafiği) ve Sparkline (mini fiyat grafiği) saf SVG kodla JavaScript içinde üretilir, harici kütüphane kullanılmaz.

CSS Yükleme Sorunu ve Çözümü

Vite, <link rel="stylesheet"> etiketleri üzerindeki CSS dosyalarını JavaScript HMR modülüne dönüştürür. Bu nedenle özel stiller tarayıcıda sıfır kural olarak görünüyordu. Çözüm: import appCss from './app.css?raw' ile CSS, JavaScript string'i olarak içe aktarılır ve IIFE içinde <style> etiketi olarak DOM'a enjekte edilir. Bu yöntem incognito dahil tüm tarayıcılarda çalışır.

2. Kural Tabanı — IF-THEN Kuralları

18 adet çıkarım kuralı, kullanıcı cevaplarına göre tetiklenir ve her laptop için puan değiştirir:

RULE gaming_gpu_match:
  IF   answers.gaming = "serious" AND laptop.hasGpu = true
  THEN score += +30 · reason: "Dedicated GPU matched for serious gaming"
RULE battery_priority:
  IF   answers.battery = "yes" AND laptop.batteryHours < 10
  THEN score += −20 · reason: "Battery life below requirement"
RULE vm_ram_check:
  IF   answers.vmUse = "yes" AND laptop.specs.ram < 16
  THEN score += −15 · reason: "Insufficient RAM for virtualization"

3. İleri Zincirleme (Forward Chaining)

Çıkarım motoru, kullanıcı cevaplarını (working memory) alarak tüm 18 kuralı sırayla değerlendirir. Koşulu sağlayan her kural "ateşlenir" (fires) ve laptop puanı değiştirilir.

Puan Hesaplama Formülü:

baseScore = 50 (tüm laptoplar buradan başlar)
+ Σ(ateşlenen kuralların puan katkıları)
+ budgetProximityBonus — Bütçeye yakınlık bonusu (max +20)
+ (valueScore − 7) × 3 — Para/performans oranı bonusu

finalScore = max(0, hesaplananPuan)

HOT Etiketi Neden Görünür?

HOT etiketi, aşağıdaki kriterlerden en az ikisini karşılayan laptoplara manuel olarak atanmıştır:

  • 🔥 Değişim yüzdesi +3% ve üzerinde
  • 🔥 Bilgi tabanındaki valueScore ≥ 8.5 (para/performans oranı yüksek)
  • 🔥 Expert System'in en sık önerdiği modeller arasında yer alıyor
  • 🔥 Geniş kullanıcı kitlesine hitap eden çok amaçlı (versatile) laptop

Örnek: MacBook Air M4, hem en yüksek pil ömrüne (+18 saat) hem de en hafif tasarımlardan birine (%1.24kg) sahip olduğu için HOT seçilmiştir.

NEW Etiketi Neden Görünür?

NEW etiketi, 2025–2026 yılı içinde piyasaya çıkan modellere uygulanmıştır. Framework 13 AMD (2026 modeli) ve MacBook Neo A18 bu kategoridedir. Piyasaya yeni giren modeller genellikle en yüksek talep artışını yaşar (Framework: +12.4%, MacBook Neo: +18.5%).

Fiyat Titreme (Price Flicker) Mantığı

Sayfa açıldıktan 4 saniye sonra başlayan, her 3.5–5.5 saniyede bir tekrar eden fiyat güncellemesi şu formülle hesaplanır:

delta = (Math.random() - 0.5) × basePrice × 0.008
newPrice = Math.round(basePrice + delta)

// Örnek: €1,199 fiyatlı laptop için
delta = ±(1199 × 0.008) = ±€9.59 aralığında rastgele
newPrice ∈ [€1,189 … €1,208]

Yeni fiyat eskiden yüksekse yeşil flash (priceUp), düşükse kırmızı flash (priceDown) animasyonu tetiklenir. 600ms sonra animasyon temizlenir.

Ticker Kaydırma Tekniği

CSS animasyonu yerine JavaScript requestAnimationFrame kullanılmasının sebebi şudur: CSS animasyonları servis worker cache'i tarafından etkilenebilir ve bazı tarayıcılarda sekebilir. RAF ise her kare için GPU'dan bağımsız, tarayıcının paint döngüsüyle senkronize çalışır.

// Sonsuz döngü için tüm öğeler klonlanır
origItems.forEach(item => track.appendChild(item.cloneNode(true)));

// Her frame'de 0.6px sola kaydır
pos += 0.6;
// Yarı noktaya gelince başa sar (seamless loop)
if (pos >= track.scrollWidth / 2) pos -= track.scrollWidth / 2;
track.style.transform = `translateX(-${pos}px)`;
